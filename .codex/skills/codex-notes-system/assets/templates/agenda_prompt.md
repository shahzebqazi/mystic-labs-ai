# Agenda Application Generator Prompt

Create a small, fast GUI app that generates intelligent agendas from buildlog.txt metadata.

## Core Requirements
- Minimal dependencies, fast startup, low memory.
- Parse only the current project's buildlog.txt.
- Generate multiple agenda types dynamically.

## Agenda Types
- Todo list generation
- Calendar integration
- Changelog generation
- Journal creation
- Grocery list abstraction

## Output
Build a runnable app that:
1. Reads buildlog.txt
2. Determines project needs
3. Generates agendas
4. Provides a clean interface
5. Syncs changes bidirectionally
