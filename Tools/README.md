**Style Guide**: This directory follows the style guidelines defined in `Map.md`. Refer to that document for complete symbolic language definitions, AI model configurations, and architectural patterns.

## Overview
The Tools directory contains **meta prompts** for specialized AI assistants and task management systems. Every file is a complete, self-contained AI configuration that can work independently or be combined with other prompts for enhanced functionality.
