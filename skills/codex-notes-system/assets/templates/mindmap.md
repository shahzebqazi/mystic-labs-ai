# Mind Map - AI Tool and Prompt Style Guide

## Core Principles
- Single source of truth for style and symbols.
- Changes here propagate to all prompts and tools.
- No hardcoded style in individual prompts.

## Symbolic Language

### Status
! active
@ blocked
# pending
$ completed
% paused
& cancelled
!!! urgent
@@ high priority blocked
* in review
** released
-> transition

### Priority
! critical
@ medium
# low
$ pinned
!!! emergency
@@ high
* normal
** low
-> future

### Roles
! developer
@ designer
# tester
$ PM
!!! lead
@@ architect
* QA
** DevOps
-> intern

### Progress
0% not started
25% planning
50% development
75% review
90% final polish
100% complete

## Prompt Structure

- Purpose
- Context
- Dependencies
- Core requirements
- Technical specs
- Output format
- Integration

## Update Protocol

1. Identify change.
2. Assess impact.
3. Update this file.
4. Update tools.
5. Test integration.
6. Document in changelog.
