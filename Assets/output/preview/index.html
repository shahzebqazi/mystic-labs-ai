Mystic

Magical Notes, Designs and Products

Overview

Mystic is a creative workspace for capturing notes, designing interfaces, and managing products — all in one place. It ships as a desktop app with full GUI support, a CLI for automation and scripting, and a plugin architecture for extensibility. The lo-fi pixel-art brand identity, anchored by a gold crescent moon and twinkling star field, is an invitation to let your imagination run wild.

This implementation of an AI tool is built around fun, creativity, collaboration, privacy, safety, and local-first use. Please use AI responsibly.

All brand assets (icons, logos, banners, heroes, SVGs, CSS tokens, and this preview page) are generated programmatically by the Kotlin toolchain in Assets/. Run ./gradlew run to regenerate the full set of 80 assets from brand.yaml.

Research

Cursor Brand

Mystic's visual identity draws inspiration from Cursor—the AI-first code editor by Anysphere—while adding its own personality and color. This section documents the reference design system we studied and the deliberate departures we made to give Mystic a warmer, more expressive character.

Typography

Cursor's 2025 brand refresh introduced Cursor Gothic, a custom typeface designed by Munich-based foundry Kimera in collaboration with Justin Jay Wang. Built on Kimera's Waldenburg, it blends the rationalism of Akzidenz-Grotesk with the analog warmth of Univers. Slightly more condensed and higher-contrast than its parent, it reads smoothly at both display and text sizes. A distinctive feature is logo ligatures baked into the font itself— embedding logo lockups that auto-center with surrounding text.

Cursor also pairs the typeface with Berkeley Mono for code surfaces and JJannon (a serif) for editorial content, giving it three distinct typographic voices.

Logo System

The Cursor logo extends Ben Barry's original cube design into three rendering tiers:

  • 2D flat—single-color, optimized for small sizes and favicons
  • 2.5D—flat color planes for medium applications (social cards, headers)
  • 3D—lighting and material effects for app icons and OS integrations

Subtle rounding on cube and cursor-arrow corners ensures screen clarity at every size. Each tier ships in light and dark variants.

Design Philosophy

Cursor positions itself as a "power tool for professional developers", balancing high-end technical sophistication with human warmth. Every visual decision prioritizes screen fidelity over geometric perfection—shapes are optically tuned rather than mathematically exact. The overall palette is restrained, almost monochromatic, relying on warm neutrals and transparency.

Mystic's Spin

Where Cursor keeps the palette austere, Mystic leans into personality and color. We retain the readability-first hierarchy—dark backgrounds, generous whitespace, clear type—but inject a hand-crafted, lo-fi warmth:

  • Gold accent replaces the neutral CTA tint, evoking moonlight and craft
  • Pixel-art icons & moon add tactile character versus vector minimalism
  • Rose Pink & Seafoam Green accents give the palette emotional range
  • Twinkling star fields create a living, ambient background layer
  • Hard-banded shading on the crescent moon nods to retro game art

The goal is to keep what makes Cursor's design work—the readability, the restraint, the focus on the craft of code—while giving Mystic its own expressive, imaginative voice.
