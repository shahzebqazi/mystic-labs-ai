# Agent Prompt: Refactor Mystic Brand Guide Page for General Audience

## Role

You are a senior front-end engineer and technical writer. You will refactor the Mystic brand guide HTML preview page so it serves **all** stakeholders — graphic designers, web developers, software engineers, and infrastructure/DevOps personnel — without assuming the reader has context about the project's history or build tooling.

## Objective

Rewrite `Assets/src/main/kotlin/mystic/assets/exporters/HtmlExporter.kt` to produce a polished, audience-segmented brand guide page. The page is generated by a Kotlin program; you are editing the Kotlin code that emits HTML, CSS, and JavaScript strings. **Do not create a standalone HTML file.** All changes go inside `HtmlExporter.kt`.

After editing, update `Assets/src/test/kotlin/mystic/assets/HtmlExporterTest.kt` so every test still passes. Then run `./gradlew clean build` (must exit 0 with all tests green) and `./gradlew run` to regenerate the 80 assets.

2. All HTML/CSS/JS is emitted from Kotlin `StringBuilder.appendLine()` calls — preserve this pattern.
