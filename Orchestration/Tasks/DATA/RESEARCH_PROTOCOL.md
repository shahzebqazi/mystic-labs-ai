# RESEARCH_PROTOCOL -- Structured Research for AI Agents

## Overview

Follow this protocol for research tasks, whitepapers, and evidence-based analysis. Ensures logical structure, proper sourcing, and clear conclusions. Use for WHITEPAPER and any research deliverables.

## Phases

1. **Hypothesis** — State the central claim or question.
2. **Literature** — Minimum 5 peer-reviewed sources.
3. **Argument** — Build logical argument with explicit premises.
4. **Counterarguments** — Anticipate and address objections.
5. **Conclusion** — Summarize findings and limitations.
6. **Glossary** — Define technical terms used.

## Source Requirements

- Minimum 5 peer-reviewed sources (journals, conferences, formal reports).
- Prefer recent (e.g. last 5 years) unless seminal.
- For each source, note:
  - Methodology
  - Sample size / scope
  - Limitations
  - Relevance to hypothesis

## Argument Structure

- **Premises** — Numbered, factual claims supported by sources.
- **Inference** — Clear logical steps from premises to conclusion.
- **Conclusion** — Explicit statement of what follows.

Avoid logical fallacies; flag assumptions explicitly.

## Counterarguments

List anticipated counterarguments and respond:
- Acknowledge valid points.
- Explain why they don't defeat the main argument or how they modify it.
- Cite sources where relevant.

## Source Evaluation Checklist

Per source, assess:

- Methodology (qualitative, quantitative, mixed)
- Sample size and representativeness
- Bias and funding
- Recency and applicability
- Limitations stated by authors

Only cite sources that survive this evaluation.
